# Composant Bouton

Le composant Bouton est utilisé pour afficher un élément de type bouton avec des propriétés personnalisables.

## Utilisation

```jsx

<app-button label="Cliquez-moi" size="medium" primary={false} />
```

## Props

Exemple de props :

<table>
  <thead>
  <tr>
    <th>Propriété</th>
    <th>Type</th>
    <th>Valeur par défaut</th>
    <th>Description</th>
  </tr>
  </thead>
  <tbody>
  <tr>
    <td>label</td>
    <td>string</td>
    <td></td>
    <td>Le texte du libellé du bouton.</td>
  </tr>
  <tr>
    <td>size</td>
    <td>'small' \| 'medium' \| 'large'</td>
    <td>'medium'</td>
    <td>La taille du bouton.</td>
  </tr>
  <tr>
    <td>primary</td>
    <td>boolean</td>
    <td>false</td>
    <td>Indique si le bouton est primaire.</td>
  </tr>
  </tbody>
</table>


## Exemples
### Bouton par défaut
```jsx
<app-button label="Bouton par défaut" />
```
### Bouton primaire
```jsx
<app-button label="Bouton primaire" primary={true} />
```
### Bouton de petite taille
```jsx
<app-button label="Bouton de petite taille" size="small" />
```
### Bouton de grande taille
```jsx

<app-button label="Bouton de grande taille" size="large" />
```
